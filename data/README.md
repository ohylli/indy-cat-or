# data/

Datasets and embedding caches. Contents are gitignored (see `.gitignore`) —
large, reproducible, and not meant for version control:

- negative datasets (Oxford-IIIT Pet, look-alike long-haired cats)
- `embeddings/` — cached embedding vectors

Indy's own photos live in `images/indy/` (tracked). Indy's *gallery
embeddings*, once we want them committed, can go in a tracked subfolder added
explicitly later.
