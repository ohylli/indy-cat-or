# experiments/

Throwaway, exploratory one-offs — the "treat each data addition as a measured
experiment" scratch space. Free to be messy; delete freely. They import the
stable `indycat` core, never the reverse.

Tracked in git (committed, shared dependency set) so there's a record of what
was tried across the evaluation staging. Prefix with a number or date to keep
them ordered, e.g. `01_baseline_oxford.py`. Promote anything worth keeping into
`scripts/`.
